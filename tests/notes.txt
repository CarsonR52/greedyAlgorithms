python3 src/cache_simulator.py data/ex.in

Test 2, Cache size 1:
Run:
py src/cache_simulator.py data/test2.in

1 6
1 2 1 2 1 2

With a cache size of just 1, every time the requested item changes, old one must be evicted logically

Expected output:
FIFO  : 6
LRU   : 6
OPTFF : 6


Error handling checks

1. Invalid cache size
Input:
0 4

Expected behavior:
Program should print an error that cache size must be at least 1

2. Wrong number of requests
Input:
3 5
1 2 3 4

Expected behavior:
Program should print an error saying the expected number of requests does not match
